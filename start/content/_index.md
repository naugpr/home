# start

This site was created with **HugoBlox Studio** using the **Startup Landing Page** template.

## Getting Started

1. **Install Dependencies**
   ```bash
   pnpm install
   ```

2. **Start Development Server**
   ```bash
   pnpm run dev
   ```

3. **Build for Production**
   ```bash
   pnpm run build
   ```
